# 纯HTML博客

本博客为纯HTML+CSS+JS静态站点，适合直接部署在 GitHub Pages，无需任何构建工具。

## 目录结构
- index.html：主页，展示欢迎语和文章列表
- posts/hello-world.html：示例文章
- static/css/style.css：全站样式
- static/js/main.js：全站脚本
- 404.html：自定义404页面

## 部署方法
1. 将本项目所有文件推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages，发布根目录（/）
3. 访问你的博客主页即可

## 文章添加
- 复制 posts/hello-world.html，修改内容即可新增文章
- 在 index.html 的文章列表中添加新文章链接
